### Final Project – Autonomous Drone
- **Title**: Autonomous Drone with Waypoint Navigation
- **Team**: Niloy Kumar Mondal (2105044), Raihan Rashid (2105046), Jonayed Mohiuddin (2105060)
- **Hardware**: Px4 flight controller, GPS module, Sensors, ESC/motors
- **Features**:
  - GPS waypoint navigation
  - Autonomous takeoff/landing
  - Obstacle avoidance (ultrasonic/IR)
  - Live telemetry via Bluetooth/WiFi
  - Battery monitoring and RTH (Return to Home)
- **Software**: Ardupilot
